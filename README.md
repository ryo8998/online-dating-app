# アプリ名
# Online-dating-app 
Recurision Vue 課題


## アプリ概要 
- コンタクトできるユーザーを一覧で表示します
- 男性・女性でユーザー一覧フィルターできます
- ユーザー一覧を年齢（若い順)でソートできます
- ユーザー一覧からユーザーを選択し、その情報が確認できます
- 気に入ったユーザーを選んでチャットができます（相手の返答は固定です）
- 一度会話したユーザーの一覧がMessage listページから確認できます
